# Dark Mode Toggle for Landing Page + App

## Files to modify
1. `src/App.tsx` — wrap providers with `ThemeProvider` from next-themes
2. `src/components/ThemeToggle.tsx` — **new file**, Sun/Moon toggle using `useTheme()`
3. `src/pages/Landing.tsx` — add toggle to desktop nav + mobile menu + dark mode CSS overrides for `LP_VARS`
4. `src/components/Topbar.tsx` — add toggle next to language switch

## Implementation

### 1. App.tsx
- Import `ThemeProvider` from `next-themes`
- Wrap the provider tree with `<ThemeProvider attribute="class" defaultTheme="system" enableSystem>`

### 2. ThemeToggle.tsx (new)
- Small button with Sun/Moon icons from lucide-react
- Uses `useTheme()` from next-themes
- Cycles: system → light → dark
- Animated icon swap with framer-motion

### 3. Landing.tsx
- Import `useTheme` from next-themes
- Add ThemeToggle to desktop nav (between nav links and Sign in)
- Add ThemeToggle to mobile menu
- Add `.dark` overrides for all `LP_VARS` in the `<style>` block:
  - `--night` → dark blue-gray bg
  - `--night-2` → darker card
  - `--moon` → light text
  - `--moon-dim` → muted light text
  - `--line` → dark border
  - `--cream` → dark section bg
  - `--ink` / `--ink-soft` → light text variants
  - Product tour mock backgrounds (white → dark)

### 4. Topbar.tsx
- Import ThemeToggle
- Place it right before the language toggle button
